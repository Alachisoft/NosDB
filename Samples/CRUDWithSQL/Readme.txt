============================================
PREREQUISITES FOR RUNNING SAMPLE APPLICATIONS
============================================

Before the sample application is executed make sure that:

** The northwind sample data has been imported into the collections as explained in 
    %InstallationDir%\NoSDB\samples\data\json\readme.txt

** The connection strings in the app.config files of samples have been changed according to the configurations made during import. For example, changes to the database name or configuration server port.
